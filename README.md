# Daily Productivity Scheduler

## Site Link
- https://alenicakariz.github.io/daily-productivity-scheduler/

### About the Project
A web application that solves the 0/1 knapsack problem to help you optimize your daily tasks and maximize productivity within your available time.

### Problem
You have limited free time (e.g., 4 hours) but many tasks to complete (study, exercise, laundry, etc.). Each task takes time and has a priority level. 
How do you choose which tasks to do to get maximum value? This is the classic 0/1 knapsack problem.

### Solution
The app uses a brute-force algorithm to test all possible task combinations and find the optimal schedule. 
It shows you which tasks to do to maximize your productivity value while staying within your time limit.

### Limitations
The brute-force approach has O(2^n) time complexity, so it's only practical for small problem sizes. 
For larger problems, dynamic programming or approximation algorithms would be better.
